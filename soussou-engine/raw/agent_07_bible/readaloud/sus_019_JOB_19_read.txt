Ayuba.
19.
Ayuba naxa a yaabi, 
«Wo fama n tɔɔrɔde han tɛmui mundun? Wo n bɔɲɛ halakima wo xa masenyi ra han tɛmui mundun? 
A dɔxɔ fu nan yi ki wo n konbima, n tɔɔrɔfe mu wo yaagima? 
Xa nɔndi na a ra, n na yunubi kui, awa, na yunubi luma n tan nan ma. 
Xa wo wama a masenfe n bɛ a wo fisa n bɛ, xa wo wama n ma tɔɔrɛ findife n ma yunubi ra, 
wo xa a kolon xɔnɛ naxan na n ma, na fatanxi Ala nan na, a tan nan n suxuxi a xa yɛlɛ ra. 
Xa n sa gbelegbele n ma xɔnɛ xa fe ra, n mu yaabi yo sɔtɔma. Xa n sa wa kiiti xa sa, kiiti tinxinxi yo mu sama. 
Ala bara kira ibolon n ya ra, n mu nɔma dangide, a bara n ma kira ifɔɔrɔ n ma. 
A bara n xun nakana, a bara binyɛ birin ba n ma. 
A bara n ma fe birin nabira. A bara n xaxili tide tala alɔ sansi bili talama ki naxɛ. 
A bara xɔnɔ n ma a ɲaaxi ra, n bara findi a yaxui ra. 
A xa geresoe bara n nabilin, e na n gerefe, n mu nɔma n gide e ma. 
A bara n ngaxakerenyie makuya n na, n kolonmae bara yensen yɛ. 
N dɔxɔbooree bara n nabɛɲin, n dɛfanbooree bara n nabolo. 
N ma xɔɲɛe nun n ma konyie, nee mu n kolon sɔnɔn. 
N na n ma konyi xili, hali n a mayandi, a mu n ma xili ratinma. 
N dɛ xiri bara raɲaaxu n ma ginɛ ma, n ma fe mu rafan n ngaxakerenyie ma. 
Dimɛe yati yoma n ma. 
N gundo xiri boore bara n xɔn, n xanuntenyie bara n yanfa. 
N fate birin bara ba n ma, n xɔrie mini kɛnɛ ma. Sese mu luxi sɔnɔn. 
N booree, wo kinikini n ma, yandi, wo kinikini n ma, barima Ala nan n bɔnbɔxi. 
Munfe ra wo n tɔɔrɔma alɔ Ala a rabaxi ki naxɛ? Wo mu nu wasa sinden?» 
«Xa n ma wɔyɛnyie sɛbɛxi buki kui nu, 
xa e masolixi gɛmɛ ma wure ra nu, na tɛmui e buma nɛ nu abadan. 
Kɔnɔ n tan a kolon n nakisima ɲiɲɛ na a ra. A fama wɔyɛnyi dɔnxɔɛ falade duniɲa ma. 
N fate na gɛ bɔrɔde, n tan yati fama nɛ Ala tode. 
N tan nan yati fama a tode n yae ra, a mu mataalima n bɛ. Na kui n bɔɲɛ sɛɛwama nɛ a fanyi ra. 
Wo tan naxee maxɔrinyi tima, ‹Won a tɔɔrɔfe munfe ra?› Wo fama n ma nɔndi kolonde. 
Wo xa gaaxu santidɛgɛma ya ra, barima ɲaxankatɛ wo sɔtɔma nɛ wo xa fe ɲaaxie ma. Na kui wo fama a kolonde a kiitisa nde na.»
