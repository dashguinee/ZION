Yabura.
77.
Asafi xa bɛɛti bɛɛtibae xa mangɛ Yedutun bɛ 
N nan n xui ratefe Ala nan ma. A a tuli matima nɛ n xui ra. 
Lɔxɔ xɔrɔxɔɛe na fa n ma, n nan n Marigi xilima nɛ. Kɔɛ ra, n nan n bɛlɛxɛ itala a bɛ, kɔnɔ n mu madundui sɔtɔma. 
N nan n natu Ala ma, n wa n bɔɲɛ ma. N nan n maɲɔxun a ma, limaniya ba n yi. 
I mu tinma n xa xi. N xaxili ifuxi, ɲan n mu nɔma wɔyɛnde. 
N nan n maɲɔxunma waxati dangixie ma. 
N nan n natuma nee ma n ma bɛɛtie saabui ra. N nan n yɛtɛ maxɔrin, 
«N Marigi mɛɛma nɛ ibunadama ra abadan? A mu hinnɛma mixie ra sɔnɔn? 
Danyi na a xa fanyi ma? A tondima a xa laayidie kamalide? 
A bara nɛɛmu a xa hinnɛ ma? A xa xɔnɛ bara nɔ a xa kinikini ra? 
N ma tɔɔrɛ kui a luma n bɛ alɔ Ala Xili Xungbe Kanyi na masarafe.» 
A lanma n xa ratu Alatala xa walie ma, a xa kaabanakoe a naxee rabaxi tɛmui dangixi. 
N nan n xaxili rasigama nɛ i xa kɛwali birin ma, n nan n maɲɔxunma nɛ i xa walie ma. 
Ala, i xa kira sɛniyɛn. Ala gbɛtɛ mu na, i maniyɛ mu na! 
I tan Ala nan kaabanakoe rabama, i fe magaaxuxie masen ɲamanɛe tagi. 
I i xa ɲama xun saraxi i sɛnbɛ nan na, Yaxuba nun Yusufu xa mamadie. 
Ala, baa xungbe to i to, a dulonyi naxa sɛrɛn. 
Nuxuie naxa ye gbegbe ragoro, galanyi naxa bula koore ma, seyamakɔnyie naxa lu alɔ tanbɛ wolixie. 
I xa sarinyi xui naxa bula, seyamakɔnyi naxa duniɲa iyalan, bɔxi naxa a imaxa, a sɛrɛn. 
I naxa kira rabɔɔ baa ma, i dangi ye xungbe tagi, kɔnɔ i fɔxi mu to. 
I naxa i xa ɲama raɲɛrɛ alɔ xuruse gɔɔrɛ Annabi Munsa nun Haruna saabui ra.
