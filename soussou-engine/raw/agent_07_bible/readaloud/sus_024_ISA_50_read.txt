Annabi Esayi.
50.
Alatala xa masenyi nan ya: «N kɛɛdi mundun sɛbɛxi n xa mɛɛ wo nga ra? N sare mundun sɔtɔxi wo matife ra? Wo matixi wo yɛtɛ xa yunubie nan ma fe ra. N mɛɛxi wo nga ra a xa matandi nan ma. 
Munfe ra wo mu n yaabima? Munfe ra wo mu tima n ya ra? Wo a maɲɔxunxi nɛ n bɛlɛxɛ mu fata wo fulunde? Wo a maɲɔxunxi nɛ n sɛnbɛ mu nɔma wo rakiside? N ma wɔyɛn keren nɔma baa xarade. N nɔma xure findide gbengberenyi ra, yɛxɛe xa faxa, e xa bɔrɔ. 
N fata koore rafɔɔrɔde, alɔ mixi na kaaɲɛ dugi ragoroxi a ma.» 
N Marigi Alatala bara masenyi so n yi ra naxan nɔma mixi taganxi malide. A n na naxunuma gɛɛsɛgɛ birin, a n tuli ibama, alako n xa n tuli mati a ra alɔ xarandi fanyi. 
N Marigi Alatala bara n tuli rabi, n tan mu tondima birade a fɔxɔ ra. 
N nan n fari tima nɛ n yaxuie xa n bɔnbɔ. N na n xɛrɛ fan tima nɛ e xa n dɛ xabe matala. N mu n ya tagi nɔxun e xa konbie ma, n mu tondi e xa e dɛ ye bɔxun n ma. 
N Marigi Alatala to n mali, n mu yaagi, n mu n gi. N luma nɛ alɔ fanye, barima n a kolon n mu luma yaagi kui. 
Naxan n gbe ɲɔxɔma a na n sɛɛti ma. Nde wama n sɔnxɔfe? Won xa wɔyɛn. Nde wama n kalamufe? Kiiti xa sa. 
N Marigi Alatala nan n malife. Nde nɔma a ra? E xa kalamui kanama nɛ alɔ dugi fori teese naxan donma. 
Nde gaaxuma Alatala ya ra wo ya ma, a Ala xa konyi xui suxu? Naxan birin na ɲɛrɛfe dimi xɔɔra naiyalanyi xanbi, a xa a xaxili ti Alatala xili ra, a la a ra. 
N xa a fala wo bɛ, wo tan naxee tɛ radɛxɛma, wo fa tanbɛ raxaɲɛnma gere xili ma, N fama nɛ wo tɔɔrɔde a xɔrɔxɔɛ ra.
