Samuweli II.
22.
Alatala to Dawuda ratanga Sɔlu nun a yaxuie ma, Dawuda naxa yi bɛɛti ba Alatala bɛ. 
A naxɛ, «Alatala nan n ma fanye ra, Alatala nan n ma yire makantaxi ra, Alatala nan n nakisima ra. 
N Marigi Ala luxi alɔ gɛmɛ yire n lingira sɔtɔma dɛnnaxɛ. A luxi alɔ wure lefa n bɛ, alɔ sɛnbɛma naxan n nakisima, alɔ taa makantaxi n sabatima dɛnnaxɛ. I tan, n nakisima, i n natangama n yaxuie ma. 
Ala tantu! N to i xili, i bara n natanga n yaxuie ma. 
Faxɛ nu bara wa nɔfe n na, alɔ mɔrɔnyie naxee dusuma mixi xun na. N nu bara kɔntɔfili gbaloe ra, 
faxɛ luuti nu bara n xiri, faxɛ yɛlɛ nu bara itala n ya ra. 
N ma tɔɔrɛ kui, n bara Alatala xili, n bara n Marigi Ala maxandi. A bara n xui mɛ a xa hɔrɔmɔlingira kui, n wa xui bara a tuli li. 
Awa, koore nun bɔxi naxa imaxa, e sɛrɛn Ala xa xɔnɛ ya i. 
Tuuri nu minima a ɲɔɛ i, tɛ nu minima a dɛ i, tɛ wole nu kasanma. 
A naxa koore walaxɛ ɲɛnkɛn, a goro nuxui binye fari. 
A naxa ti malekɛ fari, a a ɲɛrɛ koore ma foye fari. 
Nuxui ifɔɔrɔxie nu na a rabilinyi, e rafexi ye ra. 
Tɛ wolee nu minima a xa yanbɛ ya ra. 
Alatala xui nu luxi alɔ se sarinyi. Koore Mangɛ naxa a xui ramini. 
A naxa seyamakɔnyie woli alɔ xalie, e xa n yaxuie keri. 
Baa bunyi naxa maba, bɔxi xɔri naxa makɛnɛn, Alatala ɲɔɛ foye to mini. 
A naxa a bɛlɛxɛ itala keli koore ma, a n suxu, a n namini baa ye xɔɔra. 
A naxa n natanga n yaxui sɛnbɛmae ma, n xɔnnante naxee sɛnbɛ gbo n bɛ. 
E nu n gerema n xunsin lɔxɔɛe kui, kɔnɔ Alatala nu findixi n malima nan na. 
A naxa n namini, a naxa n nakisi a xa xanunteya xa fe ra. 
Alatala n suxu n ma tinxinyi nan ma, a n ma sɛniyɛnyi sare ragbilen n ma, 
barima n nu ɲɛrɛma Alatala xa kira nan xɔn, n mu n baxi n Marigi Ala xa kira ma alɔ mixi ɲaaxie. 
N birama a xa yaamarie fɔxɔ ra, n mu gbilenma a xa sɛriyɛ fɔxɔ ra. 
N tinxin nɛ a bɛ, n fe kobi birin lu na. 
Na kui Alatala naxa n ma tinxinyi nun n ma sɛniyɛnyi sare ragbilen n ma. 
Mixi naxan tima i sɛɛti ma, i fan tima na kanyi nan sɛɛti ma. Mixi naxan mu findi filankafui ra i mabiri, i fan mu i xa laayidi kanama na kanyi bɛ. 
Mixi naxan sɛniyɛn i ya xɔri, i fan i xa sɛniyɛnyi masenma na kanyi bɛ. Mixi naxan fe kanama, i fan fata na sare ragbilende a ma. 
I mixi magoroxie itema nɛ, i yɛtɛ igboee magoroma nɛ. 
I tan Alatala nan na n ma naiyalanyi ra. Alatala nan dimi iyalanma n bɛ. 
Xa i na n fɔxɔ ra, n mu gaaxuma sɔɔri gali ya ra. Xa n Marigi Ala na n fɔxɔ ra, n suusama nɛ gere ra. 
Ala xa kira kamalixi. Ala xa masenyi sɛniyɛnxi. A findixi makantɛ nan na mixi bɛ, naxan a yɛtɛ taxu a ra. 
Nde na Ala ra xa Alatala mu a ra? Nde na ibunadama xa fanye ra xa muxu Marigi Ala mu a ra? 
Ala nan na n ma yire makantaxi ra, naxan kira fanyi yailanma n bɛ. 
A n malima n ma ɲɛrɛ kui, alɔ a xeli malima ki naxɛ a xa ɲɛrɛ kui geyae fari. 
A n malima n ma gere kui, alɔ sɛnbɛma naxan xali wure gbeeli daaxi italama. 
I findixi n ma kantama nan na, i n xun nakelima. 
I n naɲɛrɛma a fanyi ra, n ma kira mu findima tɔɔrɛ ra n bɛ. 
N na n yaxuie kerima, n e sɔntɔma. N mu gbilenma e fɔxɔ ra fo n e birin faxa. 
N e faxama nɛ, n e bɔnbɔma nɛ. E mu kelima sɔnɔn, e luma n bun ma abadan. 
I sɛnbɛ fima n ma gere kui, i n yaxuie ragoroma n bun ma. 
I n yaxuie kerima n ya ra, i n xɔnnantee ralɔɛma nɛ. 
E e ya ragema nɛ, kɔnɔ e rakisima mu na! E Alatala xilima nɛ, kɔnɔ a mu e yaabima! 
N e luntunma nɛ e findi xube ra, n e iboronma nɛ alɔ boora. 
I bara n natanga n ma ɲama xa ɲɔxɔnsɔsɔɛ ma, i bara n findi ɲamanɛe xa mangɛ ra, i bara sie lu n sagoe n mu naxee kolon. 
Si gbɛtɛe n matɔxɔma, e n ma yaamari birin suxuma. 
Si gbɛtɛe mu suusama sɔnɔn, e minima e xa yire makantaxie kui gaaxui ra. 
Kisi na Alatala bɛ, tantui na a bɛ naxan luxi alɔ fanye n bɛ, a xili xa matɔxɔ. 
Tantui na Ala bɛ, naxan n gbe ɲɔxɔma, naxan sie luma n sagoe, 
naxan n bama n yaxuie yi ra, naxan n natema yaxuie xun ma, naxan n bama mixi ɲaaxi bɛlɛxɛ i. 
Na nan a ra, n i matɔxɔma sie tagi, Alatala! N bɛɛti bama i xili ra. 
A bara a xa mangɛ sugandixi rakisi, a mu gbilen na mangɛ fɔxɔ ra, naxan findixi Dawuda nun a bɔnsɔɛ ra abadan.»
