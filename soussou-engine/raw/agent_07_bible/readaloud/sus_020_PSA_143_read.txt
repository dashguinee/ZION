Yabura.
143.
Dawuda xa bɛɛti Alatala, i tuli mati n ma maxandi ra, i xa kinikini n wa xui ma. I xa n ma duba suxu i xa dugutɛgɛɲa nun i xa tinxinyi xa fe ra. 
I naxa i xa konyi makiiti, barima daali yo mu tinxin i ya i. 
N yaxui birama nɛ n fɔxɔ ra, a n boronma nɛ bɔxi ma, a n luma nɛ faxɛ niini bun ma alɔ faxamixie. 
Na kui n xaxili bara ifu. Limaniya bara ba n yi. 
N natuma nɛ lɔxɔɛ dangixie ma, n na n xaxili rasigama i xa wali birin ma, n na n ɲɔxɔ rasiga i xa daalie ma. 
N na n bɛlɛxɛ italama nɛ i bɛ, n nii i fenma nɛ alɔ gbengberenyi ye fenma ki naxɛ. Ɲɛngi rate. 
Alatala, yandi, i xulun n yaabife ra. Limaniya bara gɛ bade n yi ra. I naxa i kobe raso n na, xa na mu n luma nɛ alɔ faxamixie. 
Kuye na iba, i xa i xa dugutɛgɛɲa masen n bɛ, barima n xaxili tixi i tan nan na. N ɲɛrɛ kira masen n bɛ, barima n xaxili tixi i tan nan na. 
Alatala, n natanga n yaxuie ma, barima n kantari sɔtɔma i niini nan bun ma. 
N xaran i waxɔnfe rabafe ra, barima i tan nan na n Marigi Ala ra. I Xaxili Sɛniyɛnxi xa n ti kira fanyi xɔn ma. 
Alatala, i xa n nakisi i xili xa fe ra, i xa n ba tɔɔrɛ kui i xa tinxinyi xa fe ra, 
i xa n yaxuie dɛ balan i xa dugutɛgɛɲa xa fe ra, i xa n gerefae xun nakana, barima i xa konyi nan lanxi n ma.
