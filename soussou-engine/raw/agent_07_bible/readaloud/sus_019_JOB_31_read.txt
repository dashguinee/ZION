Ayuba.
31.
«N natɛ tongo nɛ, n naxa yabu ginɛdimɛdi ra. 
Ala munse ɲanigema n bɛ koore ma? Ala Sɛnbɛ Kanyi kɛ mundun fima n ma kelife ra a ma? 
Setareɲa mu ragataxi mixi ɲaaxi xa bɛ? Ɲaxankatɛ mu tinxintare xa lima? 
Ala mu nu n fɔxi birin matoma xɛ? A mu nu n ma santongoe birin kɔntima xɛ?» 
«Xa n bara n ɲɛrɛ wule kui, xa n bara mixi madaxu, 
Ala xa n maniya sikeeli tinxinxi ra. A n ma tinxinyi toma nɛ. 
Xa n bara kira tinxinxi bɛɲin, xa n bɔɲɛ bara bira n ya waxɔnfe fɔxɔ ra, xa n bara n manɔxɔ fe ɲaaxi ra, 
mixi gbɛtɛe xa n ma sansi sixie bogi don, n ma sansi sixie birin xa matala bɔxi ma. 
Xa ginɛ xɔli bara n suxu, xa n bara bira n dɔxɔboore xa ginɛ fɔxɔ ra, 
n ma ginɛ xa bande ɲin xɛmɛ gbɛtɛ bɛ, a tan nun na xa kafu, 
barima yɛnɛ findixi fe ɲaaxi nan na, kiitisae naxan makiitima a xɔrɔxɔɛ ra. 
Yɛnɛ, tɛ nan a ra naxan sigama ganyi ti ra han yahannama, a mixi harige birin xun nakana.» 
«Xa n mu findi mixi tinxinxi ra nu n ma konyie mabiri, muxu to nu galanbuma, 
n Ala yaabima di nu? N nan n yɛtɛ xun mafalama di nu? 
Ala keren xa mu muxu birin daaxi muxu nga tɛɛgɛ i? Ala keren xa mu muxu birin nabaloxi muxu nga xiɲɛ ra?» 
«Xa n mu setaree ki nu, xa n mu kaaɲɛ ginɛ mali nu, 
xa n keren bara n ma donse don, n mu kiridi ki, 
n naxan xuru, n findi a bɛ babɛ ra kabi a yɔrɛ ra, naxan nga kaaɲɛ ginɛ bara lu n xɔnyi kabi a tɛɛgɛxi a ma, 
xa n mu tɔɔrɔmixi ki dugi ra nu, 
na kanyi lu a mu duba n bɛ na dugi xa fe ra naxan kelixi n ma yɛxɛɛe xabe ma, 
xa n bara kiridi nde tɔɔrɔ, n to a kolonxi kiitisae nɔndi fima n ma, 
n bɛlɛxɛ xa bolon n tunki ma, n xɔrie xa magira, 
barima n gaaxu Ala xa ɲaxankatɛ ya ra a gbe ra, n mu nɔma sese ra a sɛnbɛ ya i.» 
«Xa n bara n xaxili ti n ma xɛɛma ra, xa n bara la n ma naafuli ra, 
xa n bara ɲɛlɛxin n harige xa gboe ra, n naxan sɔtɔ n ma duniɲɛigiri kui, 
xa n bara n ya ti soge nun kike xa yanbɛ ra, 
n fa nee batu n bɔɲɛ kui gundo ra, n nan n bɛlɛxɛ sunbu e xa binyɛ bun ma, 
Ala xa n ɲaxankata, barima na fe mɔɔli mu fan, n bara findi yanfante ra Ala mabiri.» 
«N bara ɲɛlɛxin n xɔnma xa tɔɔrɛ ra, n sɛɛwa a xa ɲaxankatɛ ra, 
n tan naxan mu nu tinma a dankade n dɛ ra, n xa na yunubi mɔɔli raba? 
N ma xɔɲɛe birin mu nu a falama xɛ: ‹A n dɛ wasaxi sube ra›? 
N nu biyaasilae yigiyama nɛ alako e naxa xi tande. 
N bara n ma fe ɲaaxie nɔxun n bɔɲɛ kui alɔ adamadie darixi a ra ki naxɛ? Ade. 
N bara gaaxu ɲama xa wɔyɛnyi ra, xa na mu a ra n xabilɛ xa wɔyɛnyi ra, n fa lu n xɔnyi nɔxunyi kui n mu nɔndi fala? Ade.» 
«Xa mixi na naxan tinma a tuli matide n na, n ma masenyi dɔnxɔɛ nan na ki. Ala xa n yaabi. A naxan toma n ma, a xa na sɛbɛ, 
n xa na dɔxɔ n xun ma, n xa a gbaku n kɔn ma. 
N dɛntɛgɛ sama nɛ a bɛ, alɔ mixi dɛntɛgɛ sama mangɛ bɛ ki naxɛ. 
Xa n ma bɔxi nde bara xɔnɔ n xili ma, xa mixi nde ya ye bara silɔn na, 
xa n bara donse nde don n mu a rawalima sare fi, xa n bara a kanyi tɔɔrɔ, 
mɛngi naxan na n ma bɔxi ma, na xa findi tunbe ra, n ma fundenyi xa findi ɲooge ɲaaxi ra.» Ayuba xa masenyi dɔnxɔɛ nan na ki.
