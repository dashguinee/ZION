Yabura.
38.
Dawuda xa maxandi bɛɛti 
Alatala, i naxa n ɲaxankata i xa xɔnɛ kui, i naxa n halaki i xa bɔɲɛte kui. 
I bara n maxɔnɔ i xa tanbɛ ra, i bara n nabira. 
Yalanyi yo mu na n bɛ i xa xɔnɛ kui. N ma yunubi bara a niya n fate birin xa tɔɔrɔ. 
N ma yunubie bara gbo n xun ma. Na kote bara binya, n mu a nɔma sɔnɔn. 
N ma fie bara gbo, e xiri bara ɲaaxu. Na birin fatanxi n ma xaxilitareɲa nan na. 
N bɔɲɛ bara kana, xunnakeli mu na n bɛ. N luma fɛɛɲɛn na sunnunyi nan kui. 
N fate birin wolenxi, yalanyi yo mu na sɔnɔn. 
N sɛnbɛ bara kana, n fate birin bara ba n ma. N bɔɲɛ tɔɔrɔxi tun. 
N Marigi, i n waxɔnfe kolon, n ma tɔɔrɛ mu nɔxunxi i ma. 
N bɔɲɛ mu saxi, n sɛnbɛ bara ɲɔn. N ya mu naiyalanyi toma sɔnɔn. 
N xanuntenyie nun n booree bara keli n xun ma n ma fure xa fe ra, n ngaxakerenyie bara e makuya n na. 
Naxee wama n faxafe, e bara yɛlɛ itala n ya ra. Naxee wama n tɔɔrɔfe, e bara n konbi, e n tɔɔɲɛgɛma tɛmui birin. 
Kɔnɔ n mu n tuli mati na wɔyɛnyie ra, n mu n dɛ rabi n xa e yaabi. 
N luma alɔ tuli xɔri, n luma alɔ boboe. 
Alatala, n xaxili tixi i tan nan na. N Marigi Ala, i tan nan gbansan nɔma n yaabide. 
N naxɛ, «I naxa tin n yaxuie xa yo n ma, e xa ɲɛlɛxin n ma tɔɔrɛ xa fe ra.» 
A gbe mu luxi n xa bira, n na tɔɔrɔfe tɛmui birin. 
N bara n ti n ma yunubi rabaxie ra, n yangoe naxee rabaxi, na fe bara n kɔntɔfili. 
N yaxuie tan kɛndɛ, e sɛnbɛ gbo. N xɔnmixie bara wuya, hali nɔndi mu na e bɛ. 
Naxee fe fanyi masarama fe ɲaaxi ra, e n tɔɔrɔma, barima n tan birama fe fanyi nan fɔxɔ ra. 
Alatala, i naxa n nabɛɲin. N Marigi Ala, i naxa i makuya n na. 
N Marigi, i xa fa mafuren n malide. N Marigi, n ma marakisima nan i ra.
