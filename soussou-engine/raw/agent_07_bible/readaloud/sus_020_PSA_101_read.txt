Yabura.
101.
Dawuda xa bɛɛti N dugutɛgɛɲa nun tinxinyi matɔxɔma nɛ. N bɛɛti bama i tan nan bɛ Alatala. 
N fe birin nabama lufe ra tinxintɔɛ xa kira xɔn. I fama n yire mun tɛmui? N na n ɲɛrɛma nɛ n bɔɲɛ fiixɛxi ra n ma mixie tagi. 
N mu n ya tima fe tinxintare ra, n mu tinma yanfante xa kira ra, n nun e tan mu lanma fe yo ma. 
N tan nun mixi ɲaaxi tagi kuya, n mu wama yunubi xɔn. 
N mixi dɛ balanma nɛ naxan a boore mafalama. N mu tinma yɛtɛ matɔxɔɛ nun yɛtɛ igboɲa ra. 
N na n ɲɔxɔ sama nɛ yi bɔxi dugutɛgɛe xɔn ma, alako e xa lu n sɛɛti ma. N wama tinxintɔɛe nan xɔn ma, e xa n mali. 
Yanfantee mu luma n ma mixie ya ma, wule falɛe mu tima n ya i. 
Gɛɛsɛgɛ birin n mixi kobie sɔntɔma nɛ, alako na mixi mɔɔli naxa lu Alatala xa taa kui.
