Yabura.
31.
Dawuda xa bɛɛti bɛɛtibae xa mangɛ bɛ 
Alatala, n bara n xaxili ti i ra. I naxa tin n xa lu yaagi kui, i xa kiiti fanyi sa n ma. 
I xa i tuli mati n na, n nakisi. I xa lu n bɛ alɔ fanye, alɔ yire makantaxi alako n xa kisi. 
N ma sɛnbɛma nun n ma kantama nan i ra. I xa n xun ti kira fanyi xɔn i xili fanyi xa fe ra. 
N yaxuie bara yɛlɛ itala n ya ra, kɔnɔ i fama n natangade na ma. 
N bara n xaxili ti i ra. Alatala Nɔndi Marigi, i bara n nakisi. 
Kuye batui mu rafan n ma, n tan, nan n taxuma Alatala nan na. 
N sɛɛwama nɛ i xa hinnɛ xa fe ra, barima i bara n ma tɔɔrɛe to, n sondonyi na ɲaxankata, i na kolon. 
I mu n soxi n yaxuie yi ra, i bara n xɔrɛya. 
Alatala, hinnɛ n na, barima n na tɔɔrɔfe. N fate birin bara ba n ma. 
N ma simaya na ɲɔnfe tɔɔrɛ kui. N ma duniɲɛigiri xɔrɔxɔ. N sɛnbɛ bara ɲɔn n ma yunubi xa fe ra. N fate bara kana 
n yaxuie xa fe ra. N bara yaagi n dɔxɔbooree xɔn ma. N booree, xa na mu a ra naxee n toma taa kui, e e gima nɛ n ma. 
E birin bara nɛɛmu n ma alɔ n ɲan bara faxa. N luxi alɔ se lɔɛxi. 
N bara n yaxuie xui mɛ. Gbaloe na n nabilinyi. E bara lan a ma e xa n faxa. 
Kɔnɔ n tan bara n taxu i ra, i tan n Marigi Alatala. 
N na i sagoe. N natanga n yaxuie ma naxee wama n ɲaxankatafe. 
I xa i xa konyi to, n nakisi i xa hinnɛ ra. 
Alatala, n na i xili, i naxa n lu yaagi kui. Ala xa mixi ɲaaxie rayaagi, Ala xa e rasabari faxɛ ra. 
Ala xa e xui radundu, barima yi wule falɛe e yɛtɛ igboma, e yo tinxintɔɛe ma. 
I xa hinnɛ gbo, i naxan nagataxi i yaragaaxuie bɛ, naxee i xa kantari fenma. 
I bara e yigiya alako fe kobi rabae naxa e suxu. I bara e ratanga wule falɛe ma. 
Alatala tantu, barima a bara hinnɛ n na n ma tɔɔrɛ kui. 
N ma gaaxui kui, n nu bara a fala, «I bara i nɔxun n ma,» kɔnɔ i bara n ma maxandi suxu. 
Danxaniyatɔɛe, wo xa Alatala xanu. Alatala togondiyatɔɛe rakisima nɛ, kɔnɔ a yɛtɛ igboe ɲaxankatama nɛ. 
Wo tan naxee wo xaxili tixi Alatala ra, wo wo sɛnbɛ so, wo xa limaniya.
