Yabura.
18.
Alatala xa konyi Dawuda xa bɛɛti bɛɛtibae xa mangɛ bɛ. A yi bɛɛti nan ba Alatala bɛ a to a ratanga Sɔlu nun a yaxui birin ma. 
Alatala, n sɛnbɛ nun n xanuntenyi nan i ra. 
N nakisima Alatala luma alɔ fanye, a findi n bɛ tɛtɛ ra. N Marigi Ala luma alɔ gɛmɛ xungbe, n yigiyama dɛnnaxɛ. Sɛnbɛma nan a ra naxan n nakisima, naxan n makantama. 
N xa a fala n xui itexi ra: Ala tantu! Ala bara n nakisi n yaxuie ma. 
Saya nu bara n nabilin, faxɛ nu bara gbaku n xun ma. 
Laaxira nu bara makɔrɛ n na, gaburi nu bara a dɛ rabi n ya ra. 
N Alatala maxandima n ma tɔɔrɛ kui, n nan n Marigi Ala xilima n xui itexi ra. A n xui mɛma a xa hɔrɔmɔbanxi kui, a a tuli matima n wa xui ra. 
Bɔxi naxa sɛrɛn, geya naxa ramaxa Ala xa xɔnɛ xa fe ra. 
A xa xɔnɛ naxa lu alɔ tuuri naxan tema, alɔ tɛ naxan minima a dɛ i, alɔ tɛ wole dɛxɛxi. 
Ala naxa kuye ibɔɔ firin na, a goro, a sanyie tixi nuxui ifɔɔrɔxi fari. 
A ti malekɛ xun ma, a ɲɛrɛ koore ma foye fari. 
A a nɔxun dimi kui, tunɛ nun nuxui ifɔɔrɔxie tagi. 
Seyamakɔnyi naxa mini a ya ra nuxuie kui. Balabalanyi nun tɛ naxa goro. 
Alatala xui naxa lu alɔ galanyi kelife koore, Ala Xili Xungbe Kanyi naxa a xui ramini, balabalanyi nun tɛ fan naxa goro. 
A naxa a xa tanbɛe woli, a a yaxuie rayensen seyamakɔnyi ya ra. 
Baa nun bɔxi naxa rabi Alatala xa xɔnɛ ya ra. 
Alatala naxa n suxu kelife koore ma, a n ba ye xɔɔra. 
A naxa n natanga n yaxui sɛnbɛmae ma, n xɔnmixi naxee sɛnbɛ gbo n bɛ. 
E naxa n gere n ma tɔɔrɛ kui, kɔnɔ Alatala naxa n natanga. 
A naxa n namini gbaloe kui a xa xanunteya saabui ra. 
Alatala bara n ma tinxinyi sare ragbilen n ma, a bara n mali n ma kɛwali sɛniyɛnxi xa fe ra. 
N bara lu Alatala xa kira xɔn, n mu n Marigi Ala bɛɲin kobiɲa ra. 
N bara a xa sɛriyɛ birin suxu, n mu tondixi a xa yaamarie rabatude. 
N bara tinxin a ya ra, n mu tin birade yunubi fɔxɔ ra. 
Alatala bara n ma tinxinyi sare ragbilen n ma, a bara n mali n ma kɛwali sɛniyɛnxi xa fe ra. 
I dugutɛgɛɲa masenma dugutɛgɛɲa kanyi bɛ. I tinxinyi masenma tinxintɔɛ bɛ. 
I sɛniyɛnyi masenma sɛniyɛntɔɛ bɛ, kɔnɔ i kɔɔta nde rabama mixi ɲaaxi ra. 
I yɛtɛ magoroe rakisima nɛ, kɔnɔ i yɛtɛ igboe tan magoroma nɛ. 
Alatala, i tan nan naiyalanyi ra n bɛ. N Marigi Ala, n ba dimi kui. 
Xa won birin na a ra, n mu gaaxuma gali xungbe gerede. N nɔma gere xɔrɔxɔɛ ra Ala xa mali saabui ra. 
Ala xa kira fan. Alatala xa masenyi tinxin. A mixie kantama naxee yigiya fenma a ra. 
Ala gbɛtɛ mu na, fo Alatala. Sɛnbɛma mu na, fo won Marigi Ala. 
Ala nan sɛnbɛ fima n ma, a n naɲɛrɛ sɛniyɛnyi kui. 
A n malima n xa lu alɔ xeli, naxan fata ɲɛrɛde yire xɔrɔxɔɛ. 
A bara n nafala gere kui, a n malima xali fanyi rawalide. 
I bara n xun nakeli, i bara n tongo i xɔnyɛ i, i bara i ɲɛngi sa n xɔn ma. Na nan a niyaxi n dariyɛ sɔtɔxi, 
n sɛnbɛ sɔtɔxi, alako n naxa bira. 
N birama n yaxuie fɔxɔ ra han n e suxu. N mu gbilenma e fɔxɔ ra fo n nɔ e ra. 
N nee rabirama nɛ han e mu nɔ kelide sɔnɔn, e fa lu n sanyi bun. 
I sɛnbɛ soma n yi ra n ma gere kui. I n yaxuie ragoroma n bun ma, 
i a niya n xa nɔ e ra. N xɔnmixie mu wɔyɛnma sɔnɔn. 
E mixie xilima e xa fa e mali, kɔnɔ mixi yo mu e rakisima. E Alatala xilima, kɔnɔ a fan mu e yaabima. 
N e luxutama nɛ e fa lu alɔ xube, foye naxan xaninma. N e boronma alɔ bɛndɛ. 
I bara n natanga ɲama murutaxi ma, i fa n findi sie xunyi ra. N mu naxan kolon na bara lu n bun ma. 
E n xui rabatuma, xɔɲɛe wɔyɛn ɲɔxunmɛ falama n bɛ. 
E mu suusama n ya ra. E minima e xa tɛtɛ makantaxi kui gaaxui ra. 
Alatala sɛnbɛ mu ɲɔnma abadan. Tantui na n ma Kantama bɛ. Tantui na n Marigi Ala Marakisima bɛ. 
Ala nan n gbeɲɔxɔma. Ala nan sie rabirama n ma nɔɛ bun ma. 
Ala nan n natangama n yaxuie ma. I tan nan n itema n yaxuie xun ma. I tan nan n natangama suutee ma. 
Alatala, n i tantuma sie tagi na birin nan ma, n bɛɛti bama nɛ i bɛ. 
A xunnakeli nun hinnɛ fima nɛ a xa mixi sugandixi ma, naxan findi mangɛ Dawuda nun a xa die ra abadan.
