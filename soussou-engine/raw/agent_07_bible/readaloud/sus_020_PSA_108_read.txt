Yabura.
108.
Dawuda xa bɛɛti 
Ala, n bara natɛ tongo n bɔɲɛ kui. N Marigi Xunnakeli Kanyi, n n xui raminima nɛ bɛɛtie ra. 
N n ma kɔrae nun n ma maxasee berema nɛ subaxɛ ma. 
N Marigi, n i batuma nɛ sie tagi, n i matɔxɔma nɛ bɛɛti ra ɲamanɛe ya ma, 
barima i xa dugutɛgɛɲa gbo dangi koore ra, i xa xanunteya ite alɔ nuxuie. 
Ala, i xunyi rakeli koore ma, i xa nɔrɛ xa lu bɔxi birin xun ma, 
alako i xanuntenyie xa kisi i sɛnbɛ saabui ra. Yandi, n ma duba suxu. 
Ala bara wɔyɛn a xa lingira sɛniyɛnxi kui, «N n xun nakelima nɛ, n Sikemi bɔxi itaxun, n Sukɔti gulunba sɔtɔma nɛ. 
N gbe nan Galadi ra, n gbe nan Manasi ra. Efirami bɔxi tan luma nɛ alɔ n xun makantase, Yudaya findi n ma munyaati ra. 
N n sanyi maxama Mowaba bɔxi ma, n n ma sankiri wolima Edon nan ma a xa findi n gbe ra. N gere ti xui mɛma nɛ Filisita bɔxi ma.» 
«Nde n xun tima taa makantaxi ra? Nde n xaninma Edon bɔxi ma? 
Ala mu a ra, naxan muxu rabɛɲinxi gere kui? 
I xa fa muxu malide, i xa muxu ba muxu yaxuie yi ra. Adamadie xa mali fufafu na a ra. 
Won fama won sɛnbɛ raminide Ala xa mali saabui ra. A tan nan fama won yaxuie iwurude.»
