## Language Information

### About Susu/Soussou

- **Native name**: Sosoxui
- **French spelling**: Soussou
- **Also known as**: Sose, Soso, Susoo
- **Language family**: Niger-Congo > Mande > Western Mande
- **Closest relative**: Yalunka
- **Total speakers**: ~2.4 million
  - Guinea: 2.2 million (mainly coastal southwest)
  - Sierra Leone: 184,000 (Northern province)
  - Senegal: 44,800
  - Guinea-Bissau: 5,340

### Writing Systems
1. Latin alphabet (primary, revised 1988)
2. Arabic script (Ajami variant)
3. N'Ko alphabet (sometimes)

### Key Grammatical Features
- Word order: SOV (Subject-Object-Verb)
- Tonal: 3 tones (high, mid, low)
- No grammatical gender
- No noun classes
- No articles (definite/indefinite)
- Plural: Add "-e" suffix
- Negation: Particle "mu"
- First person plural: Two forms (inclusive "won" vs exclusive "muxu")

## Notes on French Loanwords

Per project instructions, French loanwords are NORMAL in Susu:
- "woto" (car) - from French "auto"
- "bateau" (boat) - direct from French
- "comprendfe" (understand) - from French "comprendre"
- "lekoli" (school) - from French "l'ecole"

These should be accepted as valid Susu vocabulary.
