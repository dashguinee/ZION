Ayuba.
16.
Ayuba naxa a yaabi, 
«N bara wo xa masenyi maniyɛ gbegbe mɛ. Wo mu fata mixi madundude. 
Wo xa masenyie maniya foye ra. E ɲɔnma tɛmui mundun? N ma yaabie bara wo raxɔnɔ? 
Xa wo tan nan nu na tɔɔrɔfe, n tan fan nu nɔma wɔyɛnde nɛ alɔ wo tan, n wo tɔɔɲɛgɛ, n yo wo ma. 
Kɔnɔ n tan mu nu na rabama, n wo sɛnbɛ soma n ma wɔyɛnyie nan na nu, n dɛ wo ralimaniya.» 
«N ma wɔyɛnyi mu n ma tɔɔrɛ bama, n ma sabari fan mu a makuyama n na. 
Yakɔsi fa, n bara tagan. Ala, i bara n ma denbaya birin halaki. 
I bara n fate birin ba n ma, na bara findi seede ra n xili ma. 
Ala xa xɔnɛ bara dutun n ma, a bara n tɔɔrɔ a ɲaaxi ra, alɔ n yaxui nɔma a rabade ki naxɛ. 
N yaxuie fe ɲaaxi falama n ma fe ra, e n nayaagima, e n bɔnbɔma. E birin e malanma n xili ma. 
Ala mu n natanga fonike karinxie ma, a n so mixi ɲaaxie yi ra. 
N nu bɔɲɛsa nan kui, kɔnɔ a naxa n suxu, a n imaxa, a n xun nakana. N bara lu alɔ se a luma naxan bɔnbɔ ra. 
A xa tanbɛe n nabilinxi yire birin, e bara n bɔɲɛ sɔxɔ kinikinitareɲa kui, e n furingɛe ramini kɛnɛ ma. 
A n gerema alɔ geresoe xaaɲɛ. 
N bara sunnun donma ragoro n fate ma, n bara n xun sin bɔxi ma. 
Wa bara n yae gbeeli, saya bara makɔrɛ n na. 
Kɔnɔ n mu fe ɲaaxi yo rabaxi. N nu Ala maxandima sɛniyɛnyi nan kui. 
Bɛndɛ, i naxa dusu n wuli xun na, a lu n wa xui xa makɛnɛn yire birin.» 
«Kelife yakɔsi ma fa, n ma seede na koore ma, n xunmafalama na yire itexi. 
N booree na yofe n ma, kɔnɔ n nan n mawama Ala nan bɛ. 
N seede xa n xun mafala Ala xɔn ma, alɔ mixi a boore xun mafalama ki naxɛ, 
a xa n xun mafala barima a gbe mu luxi n xa siga, n mu gbilenma sigɛ naxan kui.»
