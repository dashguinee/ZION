Yabura.
116.
N Alatala xanuma barima a n xui mɛma. A naxa kinikini n wa xui ma. 
A to a tuli mati n na, n a maxandima nɛ n ma simaya birin kui. 
Faxɛ luuti nu bara n xiri, aligiyama tɔɔrɛ nu bara n li, nimisɛ nun kɔntɔfili nu bara nɔ n na, 
kɔnɔ na tɛmui n naxa Alatala maxandi, «Alatala, yandi, n nakisi!» 
Hinnɛnte nan Alatala ra, a tinxin. Won Marigi Ala kinikinima won ma. 
A mixi magoroxi kantama. N sɛnbɛ to ɲɔn, a naxa n nakisi. 
N xa n nii raxara, barima Alatala bara fe fanyi raba n bɛ. 
Alatala, i bara n nii ratanga faxɛ ma, i yaye ba n ya ra, i n sanyi sɛnbɛ so, 
alako n xa ɲɛrɛ i tan Alatala ya ra mixi baloxi ya ma. 
N to nu n xaxili tima i ra, n naxa a fala i bɛ, «N na tɔɔrɔfe.» 
Limaniya to ba n yi ra, n naxa a fala, «Wule falɛ nan mixi birin na.» 
Alatala, n i sare fima di, i xa hinnɛ xa fe ra? 
N i xa maratangɛ masenma nɛ ɲama bɛ, Alatala xili matɔxɔfe ra. 
N na n ma laayidie rakamalima nɛ Alatala bɛ, a xa ɲama birin ya xɔri. 
Alatala a ɲɔxɔ sama nɛ a batulae xa simaya xɔn ma. 
I tan Alatala xa konyi di nan n na, i xa konyi ginɛ xa di, i naxan nafulunxi saya luuti ra. 
N tantui sɛrɛxɛ bama nɛ i bɛ, n i xili matɔxɔma nɛ. 
N na n ma laayidie rakamalima nɛ Alatala bɛ, a xa ɲama birin ya xɔri, 
Alatala xa banxi tɛtɛ kui, Darisalamu ɲama ya xɔri. Wo Marigi matɔxɔ!
