Yabura.
26.
Dawuda gbe Alatala, i xa n ma kiiti sa, barima n ɲɛrɛ ki tinxin. N bara n yɛtɛ taxu i ra, n mu birama. 
Alatala, n bɔɲɛ mato, n sondonyi makiiti, 
barima n laxi i xa hinnɛ ra, n man n ɲɛrɛma i xa sɛriyɛ ma. 
N mu luma wule falɛe ya ma, n mu ɲɛrɛma filankafuie tagi. 
Fe kobi rabae mu rafan n ma, n tan nun mixi ɲaaxie mu luma yire keren. 
Alatala, n nan n ɲɛrɛ ki rasɛniyɛn, n man luma nɛ i xa sɛrɛxɛbade. 
N tantui bɛɛti bama, n i xa kaabanakoe masenma. 
Alatala, i xa hɔrɔmɔbanxi rafan n ma, i xa nɔrɛ na dɛnnaxɛ. 
I naxa n nii ba alɔ yunubitɔɛe nun faxatie. 
E fe ɲaaxie rabama, e kɔbiri rasuxuma tinxintareya ra. 
Kɔnɔ n tan luma tinxinyi kui. N xun sara, i xa diɲɛ. 
N ɲɛrɛ ki tinxin. N Alatala tantuma nɛ malanyie kui.
