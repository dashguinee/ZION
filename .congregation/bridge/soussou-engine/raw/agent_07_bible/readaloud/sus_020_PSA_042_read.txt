Yabura.
42.
Kora xa die xa lɔnni bɛɛti bɛɛtibae xa mangɛ bɛ 
Ala, n wama lufe i yire. N i fenma alɔ xeli ye fenma ki naxɛ. 
N hayi na Ala ɲiɲɛ ma. N a yire lima mun tɛmui? 
Yakɔsi n wama kɔɛ ra, n wama yanyi ra. Mixi n maxɔrinma, «I Marigi Ala na minden?» 
N na ratu yi fe ma, n bɔɲɛ sɛɛwama nɛ. Singe n nu ɲama raɲɛrɛma Ala xa hɔrɔmɔbanxi nan kui, e nu bɛɛti bafe ɲɛlɛxinyi kui, e tantui rasiga Ala ma. 
Munfe ra n sondonyi nimisaxi? Munfe ra n bɔɲɛ sunnunxi? N xa n xaxili ti Ala ra, n man xa a tantu a xa kisi xa fe ra. 
N Marigi Ala, n bɔɲɛ nimisaxi n yɛtɛ kan ma fe ra. Na na a ra n nan n natuma i ma Yurudɛn, n na dɛnnaxɛ yi tɛmui, Xerimon nun Misari geyae na dɛnnaxɛ. 
N ma tɔɔrɛ xun nan tun masafe. A luma alɔ susui ye naxan sinma xure ye xun. 
Alatala hinnɛma n na yanyi ra, n fan bɛɛti bama a bɛ kɔɛ ra. N Ala maxandima naxan n nabaloma. 
N bara a fala Ala bɛ, naxan sɛnbɛ gbo alɔ fanye, «I nɛɛmuxi n ma munfe ra? I n luxi tɔɔrɛ kui n yaxuie xa nɔɛ bun ma munfe ra?» 
N yaxuie n tɔɔrɔma konbi ra lɔxɔ yo lɔxɔ. E n maxɔrinma, «I Marigi Ala na minden?» 
N sondonyi nimisaxi munfe ra? N bɔɲɛ sunnunxi munfe ra? N xa n xaxili ti Ala ra, n man xa a tantu a xa kisi xa fe ra. N Marigi nan na n ma marakisima ra.
