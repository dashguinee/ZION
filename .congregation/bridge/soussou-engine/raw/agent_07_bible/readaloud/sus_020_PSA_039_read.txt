Yabura.
39.
Dawuda xa bɛɛti bɛɛtibae xa mangɛ Yedutun bɛ 
N bara a fala, «N fe birin nabama, alako n naxa yunubi raba n dɛ ra. N fe birin nabama, alako n naxa na wɔyɛnyi mɔɔli fala mixi ɲaaxie ya xɔri.» 
N ma tɔɔrɛ xungbe kui, n naxa dundu, n mu sese fala, n naxa n sabari tun. 
N bɔɲɛ nu satunma, n ma tɔɔrɛ naxa lu alɔ tɛ dɛxɛxi. Na bara a niya n xa yi wɔyɛnyi fala, 
«Alatala, a masen n bɛ n faxama tɛmui mundun, n ma simaya sigama han minden, alako n xa n sɛnbɛ xa xurunyi kolon. 
I bara n ma simaya radunke, a xɔn mu kuya sɔnɔn. Adama luma alɔ foye naxan dangima.» Ɲɛngi rate. 
Adama luxi nɛ alɔ niini, naxan minima, a fa dangi. A naafuli malanma, kɔnɔ naxan fama a sɔtɔde, a mu na kolon. 
N Marigi, n fa n xaxili tima nde ra? I tan nan gbansan nɔma n malide. 
N nakisi n ma yunubie birin ma. I naxa n nayaagi daxuie ya xɔri. 
N bara dundu, n mu sese falama, barima i tan nan na walife. 
I xa fure ba n ma, beenu i xa n naɲɔn tɔɔrɛ kui. 
I mixi ɲaxankatama a xa yunubi xa fe ra. Naxan nafan adama ma, i na bama a yi. Na fe luma nɛ alɔ kuli naxan bogise fanyi xun nakanama beenu mixi xa a matongo. Adamadi misaalixi foye nan na naxan dangima. Ɲɛngi rate. 
Alatala, n ma dubɛ suxu, i xa i tuli mati n ma maxandi ra. I xa n wa xui mɛ, barima i xa xɔɲɛ nan n na. N luma alɔ n benbae, n mu buma yi duniɲa bɛndɛ fuɲi fari. 
I xa i xa ɲaxankatɛ ragbilen n fɔxɔ ra sinden, xa na mu a ra n faxama nɛ.
